**Поиск в глубину (DFS)** — это алгоритм обхода графа, который действует по принципу «идем до упора, а там разберемся». Начав с выбранной вершины, он ныряет по ребрам вглубь структуры, пока не наткнется на тупик или уже посещенный узел. В этот момент происходит откат (backtracking) к последней развилке, где еще остались неисследованные пути. По сути, алгоритм отдает приоритет глубине, а не ширине, что делает его похожим на поиск выхода из лабиринта методом одной руки на стене.

С технической точки зрения DFS реализуется через **стек** (явно или с помощью системной рекурсии) и массив для хранения состояния вершин, чтобы не ходить по кругу. Его временная сложность составляет $O(V + E)$, где $V$ — число вершин, а $E$ — число ребер. Алгоритм критически важен для решения задач на поиск циклов, топологическую сортировку и проверку связности компонентов. Это мощный инструмент, который, однако, требует осторожности с глубиной рекурсии в очень больших графах, чтобы избежать переполнения стека.
