### **Алгоритмы:**

#### Алгоритмы поиска решения лабиринта
Алгоритмы поиска решения лабиринта представляют собой автоматизированные методы прохода. Отталкиваясь от конкретного лабиринта, можно использовать множество различных алгоритмов для нахождения его решения. В этой статье мы обсудим и проанализируем следующие хорошо известные алгоритмы поиска решения лабиринта:
- Алгоритм случайного поведения мыши ([Random Mouse Algorithm](./algorithm/Алгоритм\ случайного\ поведения\ мыши.md))
- Полный перебор/”Грубая сила” ([Bruteforce Algorithm](./algorithm/Алгоритм\ полного\ перебора.md))
- Метод следования вдоль стены ([Wall-Follower Method](./algorithm/Метод\ следования\ вдоль\ стены.md))
- Алгоритм Тремо ([Trémaux Algorithm](./algorithm/Алгоритм\ Тремо.md))
- Метод обнаружения тупиков ([Dead-End Filling](./algorithm/Метод\ обнаружения\ тупиков.md))
- Алгоритм поиска в глубину ([DFS](./algorithm/Поиск\ в\ глубину.md))
