### Технические характеристики и компоненты e-puck

|**Особенность**|**Описание**|
|---|---|
|**Процессор**|Микроконтроллер Microchip dsPIC 30F6014A @ 60 МГц (около 15 MIPS)|
|**Батарея**|Литий-ионный аккумулятор (5 Вт·ч), ~3 часа автономной работы|
|**Двигатели**|2 шаговых двигателя (20 шагов/об) с редуктором 50:1|
|**ИК-датчики**|8 датчиков (освещенность и близость препятствий до 4 см)|
|**Камера**|Цветная, макс. разрешение 640x480 (типовое: 52x39 или 640x1)|
|**Аудио**|3 всенаправленных микрофона и встроенный динамик (WAV/звук)|
|**Инерциальные датчики**|3D-акселерометр и 3D-гироскоп (оси X, Y, Z)|
|**Индикация**|8 красных светодиодов на кольце и 1 зеленый на корпусе|
|**Связь**|Bluetooth (робот-ПК, робот-робот) и ИК-порт для ДУ|
|**Управление**|16-позиционный поворотный переключатель|
|**Расширение**|Специальная шина для добавления новых модулей|
|**Программирование**|Язык C (компилятор GNU GCC), среда моделирования Webots|

| **Характеристика**                | **Значение**            |

| **Радиус колеса**                 | 20,5 мм                 |
| **Длина оси**                     | 52 мм                   |

| **Макс. скорость вращения**       | 6,28 рад/с              |

### Физические устройства робота, имена устройств e-puck

| Устройство           | Имя                                                                             |
| -------------------- | ------------------------------------------------------------------------------- |
| Моторы               | «left wheel motor» и «right wheel motor»                                        |
| датчики положения    | «left wheel sensor» и «right wheel sensor»                                      |
| Датчики приближения  | от 'ps0' до 'ps7'                                                               |
| Датчики освещенности | от 'ls0' до 'ls7'                                                               |
| Светодиоды           | от 'led0' до 'led7' (кольцо e-puck), 'led8' (корпус) и 'led9' (передняя панель) |
| Камера               | 'camera'                                                                        |
| Акселерометр         | «accelerometer»                                                                 |
| Гироскоп             | 'gyro'                                                                          |
| Датчики пола         | 'gs0', 'gs1' и 'gs2'                                                            |
| Спикер (пищалка)     | «speaker»                                                                       |
