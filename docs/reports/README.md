### **Алгоритмы:**

#### Алгоритмы поиска решения лабиринта
Алгоритмы поиска решения лабиринта представляют собой автоматизированные методы прохода. Отталкиваясь от конкретного лабиринта, можно использовать множество различных алгоритмов для нахождения его решения. В этой статье мы обсудим и проанализируем следующие хорошо известные алгоритмы поиска решения лабиринта:
- Алгоритм случайного поведения мыши ([Random Mouse Algorithm](./algorithm/Алгоритм%20случайного%20поведения%20мыши.md))
- Полный перебор/”Грубая сила” ([Bruteforce Algorithm](./algorithm/Алгоритм%20полного%20перебора.md))
- Метод следования вдоль стены ([Wall-Follower Method](./algorithm/Метод%20следования%20вдоль%20стены.md))
- Алгоритм Тремо ([Trémaux Algorithm](./algorithm/Алгоритм%20Тремо.md))
- Метод обнаружения тупиков ([Dead-End Filling](./algorithm/Метод%20обнаружения%20тупиков.md))
