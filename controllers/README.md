## Папка контроллеры
В этой папке находятся контроллеры для робота. На данный момент реализован только один контроллер - мой контроллер. В качестве чтения имеет смысл смотреть только файл [my_controller.cpp](./my_controller/my_controller.cpp). Если же вы хотите что то самостоятельно сбилдить, или поменять логику, то ворота для вас открыты - все предоставлено. Makefile позволяет сбилдить bytecode приложения. Запустите make или аналог для компиляции
- [исходный код контроллера](./my_controller)
